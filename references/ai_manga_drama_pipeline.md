# AI漫剧制作流程参考

> 基于传统微短剧工业流程与当前AI漫剧实践的综合参考，为全Agent自动化流程设计提供依据。

---

## 一、传统微短剧制作流程（6阶段）

### 1. 前期策划与立项
- 市场调研：分析目标平台（抖音1-3min/快手2-5min/B站3-10min）、受众画像、题材趋势
- 选题立项：确定题材、受众、预算档位（5万-150万）
- 交付物：项目策划书
- 把控点：选题与平台调性匹配度

### 2. 剧本开发
- IP/素材筛选 → 剧本改编
- 工程化剧本：项目总纲 → 人物圣经 → 场景圣经 → 镜头风格说明 → 分集剧本
- 每集500-800字，含场景描述、台词和情感节点
- 交付物：工程化剧本全套
- 把控点：剧情节奏、冲突密度、每集结尾钩子

### 3. 前期筹备
- 选角签约、场景勘景、服化道准备
- 交付物：拍摄计划表、服化道方案

### 4. 拍摄制作
- 按分镜/剧本现场拍摄
- 交付物：原始素材+场记单

### 5. 后期制作
- 粗剪 → 精剪 → 定剪 → 调色 → 配音/配乐 → 特效/字幕
- 交付物：成片

### 6. 审核与发行
- 内审 → 平台审核/备案 → 发行投流 → 数据复盘
- 交付物：审核通过成片、宣传物料、投流方案

## 二、AI漫剧制作流程（11环节）

```
选题选文 → 剧本开发(工程化剧本) → 角色/场景资产库构建 → 分镜脚本拆解
→ 文生图(分镜图生成) → 图生视频 → 对口型/AI配音 → 音效配乐
→ 后期剪辑/合成 → QC质检 → 发行
```

### 1. 选题选文
- 选择高传播题材（情感、悬疑、爽文逆袭）
- 评估AI可视化适配度（玄幻/古风/奇幻题材特别适合AI漫剧）
- 交付物：选题报告

### 2. 剧本开发（工程化剧本）
- **关键差异**：剧本不是台词本，而是"工程化剧本"
- 必须包含：项目总纲、人物圣经（外貌/性格/服装/发型详述）、场景圣经、镜头风格说明
- 交付物：工程化剧本全套
- 把控点：AI可执行的镜头描述清晰度；角色设定完整性（决定后续一致性）

### 3. 角色/场景资产库构建（核心差异环节）
- **AI漫剧与传统的最大区别——先建资产库，再做内容**
- 角色资产：3-5张不同角度参考图（正面/侧面/3/4角度），统一服装/发型/面部特征
- 场景资产：场景参考图库，统一光影风格
- 道具资产：纳入标准库，统一风格/质感/年代感
- 工具：即梦AI、Midjourney V7、Flux Kontext、Seedance 2.0
- 交付物：角色/场景/道具资产包
- 把控点：角色跨角度一致性；风格统一性

### 4. 分镜脚本拆解
- 每集拆解为15-25个分镜
- 每镜头时长≤10秒
- 文生图提示词格式：`{景别}，{角色外观}，{主体描述}，{环境场景}，{情绪氛围}，{视觉风格}，{光影效果}`
- 图生视频提示词格式：`{镜头描述}，{镜头运动}，{主体行为}，{背景变化}，{环境变化}`
- 关键规则：
  - 每个分镜提示词必须完整重复主角外观描述（保一致性）
  - 第一帧必须静态
  - 相邻镜头组切换场景（避免空间布局冲突）
  - 适度语义模糊（过细描述易让AI产生幻觉）
- 交付物：分镜脚本表格

### 5. 文生图（分镜图生成）
- 根据提示词批量生成分镜图
- 工具选择：豆包文生图 / Midjourney V7 / Flux Kontext
- 常见问题：违反物理现实 → 调整提示词；人物不一致 → 完整重复角色描述+参考图绑定
- 交付物：全套分镜图

### 6. 图生视频
- 分镜图作为首帧，输入图生视频提示词生成5-10秒视频片段
- **核心原则：先"稳"后"炫"**
  - 不整集一键生成
  - 一镜只做一件事
  - 先静后动，少量慢移动
- 工具选择：即梦3.0 Pro / 可灵2.1大师版 / Seedance 2.0 / Runway Gen-3
- 成本：熟练用户约5次/条（20%成片率），3分钟短片视频生成成本约1800-2000元
- 交付物：各镜头视频片段

### 7. 对口型与AI配音
- AI配音工具自动对齐口型
- 音色选择：克隆声音 / 内置音色
- 工具：即梦AI配音、剪映AI配音
- 交付物：带对口型的视频片段

### 8. 音效配乐
- 添加BGM、环境音效、动作音效
- 注意版权：使用无版权音乐库
- 交付物：完整音轨

### 9. 后期剪辑/合成
- 剪辑拼接 + 字幕 + 转场 + 调色 + 片头片尾
- 单集时长控制：1-3分钟
- 交付物：成片

### 10. QC质检
- 检查项：角色一致性、动作逻辑、音画同步、画面穿帮、合规性
- 交付物：QC报告、修改清单

### 11. 发行与合规
- 合规必须前置：分类分层审核、备案、AI商用权核查
- 发行投流：封面设计、标题优化、投流策略
- 数据复盘 → 二轮迭代
- 交付物：审核通过成片、宣传物料、投流方案

## 三、传统 vs AI漫剧 关键差异

| 维度 | 传统微短剧 | AI漫剧 |
|------|-----------|--------|
| 制作周期 | 数周到数月 | 数天到数小时 |
| 成本 | 5万-150万+ | 数千到数万 |
| 核心团队 | 10-30人 | 1-5人 |
| 资产准备 | 物理资产（服化道+场地+演员） | 数字资产包（角色/场景/道具参考图） |
| 角色一致性 | 演员天然一致 | 最大挑战，需参考图绑定+提示词重复 |
| 拍摄方式 | 实拍现场调度 | 提示词驱动：文生图→图生视频 |
| 分镜 | 可选 | 必须（提示词即分镜） |
| 关键风险 | 表演/天气/档期 | 一致性/穿帮/生成成本（抽卡） |
| 迭代方式 | 重拍/补拍 | 重新生成提示词/多抽卡 |

## 四、全Agent自动化AI漫剧流程设计参考

### Agent节点

```
Agent1: 选题Agent — 分析热点/题材/平台适配度 → 选题报告
Agent2: 编剧Agent — 生成工程化剧本（总纲+人物圣经+场景圣经+分集剧本）
Agent3: 资产管理Agent — 生成角色/场景/道具资产包
Agent4: 分镜Agent — 拆解分镜脚本，生成文生图/图生视频提示词
Agent5: 生图Agent — 批量文生图
Agent6: 生视频Agent — 图生视频
Agent7: 配音Agent — AI配音+对口型
Agent8: 音效Agent — BGM/音效
Agent9: 剪辑Agent — 拼接+字幕+转场+调色
Agent10: 质检Agent — QC检查
Agent11: 发行Agent — 合规审核+包装物料+投流策略
```

### 交付物流

```
选题报告 → 工程化剧本 → {资产包 + 分镜脚本表} → 分镜图集 → 视频片段集
→ 带配音视频 → 成片 → 质检报告 → 上线包
```

### 自动化3大核心难点

1. **角色一致性**：跨镜头/跨场景角色外观统一（参考图绑定 + 提示词完整重复）
2. **提示词质量**：精准度直接决定生成质量和抽卡次数（成本核心）
3. **镜头级生产**：不能整集一键生成，必须拆到镜头级别逐一生产再合成
