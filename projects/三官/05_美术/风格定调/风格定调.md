# 《三官》视觉风格定调

> storyboard / visual_qa 读取本文件，统一全剧画面风格。

## 总基调
- 画风：古风写实工笔，细腻厚重，偏电影质感（非卡通、非二次元）
- 画幅：9:16 竖屏（1080×1920）
- 时代：清初古装（忠于原著），服化道写实

## 色调
- 主色：低饱和青灰 + 暖橙烛光的冷暖对比
- 情绪映射：
  - 灭门（第一幕）：压抑暖黄烛光 + 阴影分明
  - 潜入（第二幕）：市井暖色 / 赵府华贵冷金交替
  - 对决（第三幕）：婚宴大红 + 内室幽暗
  - 代价（第四幕）：冷青月光、灰白，肃杀

## 光影
- 以侧光、烛光、月光为主，强调明暗对比塑造人物内心
- 关键场景（灵堂、婚宴、内室、自缢）用强对比光强化戏剧性

## 镜头语言
- 多用固定镜头 + 缓慢推进；避免快速复杂运镜（AI 视频成片率考量）
- 一镜一事，先静后动

## 负面约束（每条生成 prompt 必带）
毁容，面部扭曲，肢体变形，多余手指，模糊，现代元素，塑料感

## 一致性铁律
- 每条文生图/图生视频 prompt 必须完整重复角色描述卡的面部关键字段，不得写"同上"
- 角色服装/发型按场景（女装/男装）严格区分
